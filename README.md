# Enviador — Frontend

Projeto inicial em **React + TypeScript + Vite** com **Tailwind CSS**. Este frontend está preparado para se conectar a um backend em Django ou Flask posteriormente.

### Scripts

- npm run dev — inicia o servidor de desenvolvimento (Vite)
- npm run build — gera a build de produção
- npm run preview — pré-visualiza a build
- npm run lint — executa ESLint em `src`
- npm run format — formata o projeto com Prettier

### Começar

1. Instalar dependências:

```bash
npm install
```

2. Rodar em desenvolvimento:

```bash
npm run dev
```

3. Build de produção:

```bash
npm run build
```

### Observações
- Dependências e scripts já configurados para desenvolvimento local.
- Posso rodar `npm install` e verificar o servidor local se você quiser.
